WebLink - Link your website with your mc server

This plugin allows you to display information and perform actions on your minecraft server from your website. The API is provided by json over a TCP connection, on the default port 25511.

Requires pxnCommonMC plugin:
https://dev.bukkit.org/projects/pxncommonpluginmc
or https://www.spigotmc.org/resources/pxncommonpluginmc.107049/
